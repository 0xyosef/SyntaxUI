# Introduction

I built ansub/ui to help developers create stunning, accessible, and uniform UI through simple copy-paste actions. This system is comprised of a series of React components that are developed utilizing Tailwind CSS, I plan to include additional components, Effects, Native Animations to further enhance the system.

### Prequsites

Each section is marked with a badge like this <RequiresTailwind />
that outlines the necessary installations required for using it. To use ansub/ui, the sole prerequisite is the installation of Tailwind CSS. Other components may require shadcn/ui or additional libraries as prerequisites.

- [Tailwind CSS](https://tailwindcss.com/)
- [Next.js](https://nextjs.org/)
- [shadcn/ui](https://ui.shadcn.com)
